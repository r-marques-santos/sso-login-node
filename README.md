# 🔐 Sistema de Login com Google (SSO)

Projeto desenvolvido com Node.js utilizando autenticação via Google.

## 🚀 Funcionalidades
- Login com Google (SSO)
- Sessão de usuário
- Dashboard protegido

## 🛠️ Tecnologias
- Node.js
- Express
- Passport
- OAuth 2.0

## ▶️ Como rodar

```bash
npm install
node server.js
